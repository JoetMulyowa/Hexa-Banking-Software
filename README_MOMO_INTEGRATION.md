# MoMo Payment Integration for Apache Fineract

This project extends the Apache Fineract platform to support Mobile Money (MoMo) payment integration for loan repayment and savings deposits.

## Features

- **Loan Repayment via MoMo**: Enables loan repayments through mobile money providers
- **Savings Deposit via MoMo**: Allows savings deposits through mobile money providers
- **SMS Notifications**: Integration with AfricasTalking SMS service for transaction notifications
- **NIN Verification**: Support for National ID verification during transactions

## Technical Details

This integration is built as a module within the Apache Fineract platform, providing RESTful API endpoints for payment processing, transaction reconciliation, and notification services.
